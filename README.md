* [Learn Markdown](https://bitbucket.org/tutorials/markdowndemo)

### How do I get set up? ###

* Clone the repository
* make sure you have node installed on your machine
* run 
```
#!bash

npm install
```
* to install dependencies

```
#!bash

NODE_ENV=development node app.js
```
